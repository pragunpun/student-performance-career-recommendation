## 📌 Project Overview
This project is a machine learning-based system that analyzes student academic performance and others data to recommend suitable career paths. It helps in making data-driven decisions for education and career planning.

## 🚀 Features
- 📊 Student Performance Analysis
- 🎯 Career Recommendations
- 📄 Report Generation (PDF)
- 📈 Data Visualization using graphs
- 🧠 ML-based predictions
